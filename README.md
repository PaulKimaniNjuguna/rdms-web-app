# rdms-web-app
This project demonstrates the design and implementation of a simple relational database management system (RDBMS) using PHP, MySQL, and SQL.
It includes table creation, CRUD operations, primary and unique keys, indexing, joins, and an SQL-based interface.
The project runs locally using XAMPP (Apache + MySQL).

Technology Stack
Frontend: HTML, CSS
Backend: PHP 
Database: MySQL
Local Server: XAMPP (Apache & MySQL)
Version Control:GitHub
